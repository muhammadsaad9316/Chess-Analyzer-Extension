Real-time chess analysis Chrome extension powered by Stockfish.

## Features

- Live analysis with best move arrows on Chess.com
- Powered by Stockfish engine (runs locally)
- 100% private - no data sent to external servers

## Contents

- `INSTALL.bat` - One-click installer (run this first)
- `LAUNCH.bat` - Daily launcher (starts server + Chrome)
- `backend/` - Analysis server with Stockfish
- `extension/` - Chrome extension files
- `native-host/` - Chrome native messaging bridge

## Requirements

- Windows 10/11
- Google Chrome
- Visual C++ Runtime (included)
